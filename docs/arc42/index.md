# Architecture Documentation (ARC42)

This section documents the architecture of **fnirsi-ps-control** using the
[ARC42 template](https://arc42.org/), a pragmatic, lightweight structure for
software and system architecture documentation.

## Why ARC42?

See [ADR-001](../adr/ADR-001-use-arc42.md) for the rationale.
